Legend of Swords

About the Game

Legend of Swords is an immersive role-playing game (RPG) inspired by ancient Chinese mythology. Embark on a journey filled with mystical swords, legendary heroes, and epic battles against formidable foes. This game seamlessly blends traditional storytelling with modern gameplay mechanics to deliver an unforgettable adventure.

Key Features

  • A richly detailed world inspired by Chinese myths
  • Dynamic combat system with unique sword techniques
  • Character customization and skill upgrades
  • Engaging storyline with multiple endings
